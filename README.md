# 🚀 MeyREN Gateway

## 📌 این پروژه چطوری کار میکنه؟

اصلاً نیازی نیست سرورهای گرون‌قیمت بخرید یا درگیر کدهای پیچیده بشید. کل این پروژه روی هاست ابری و رایگان Railway اجرا میشه. فقط کافیه این پروژه رو توی گیت‌هاب خودتون کپی (Fork) کنید، با چند تا کلیک روی ریلوی راهش بندازید و تمام!

سایت ریلوی به شما یه لینک اختصاصی میده که پنل مدیریت شخصی شماست. از اونجا می‌تونید برای خودتون و دوستاتون لینک اتصال بسازید.

### خلاصه مراحل استفاده:

* 📱 برنامه‌های **v2rayNG** (برای اندروید) یا **NekoBox** (برای اندروید و ویندوز) رو روی گوشی یا کامپیوتر نصب کنید.
* 🔗 وارد پنل بشید، یه لینک VLESS بسازید و کپیش کنید.
* 📲 لینک رو وارد برنامه کنید.
* ✅ دکمه اتصال رو بزنید و تمام، شما متصل هستید!

## ⚡ شروع سریع (راه‌اندازی روی Railway)

### ۱. کپی کردن پروژه (Fork)

* اول از همه، بالای همین صفحه گیت‌هاب روی دکمه **Fork** کلیک کنید تا یه کپی از این پروژه روی اکانت خودتون ساخته بشه.

### ۲. اتصال به سایت ریلوی (Railway)

* وارد سایت [Railway.app](https://railway.app) بشید و با همون اکانت گیت‌هاب (GitHub) خودتون لاگین کنید.
* روی دکمه بزرگ **+ New Project** کلیک کنید.
* گزینه **Deploy from GitHub repo** رو انتخاب کنید و از بین پروژه‌هاتون، اسم این پروژه یعنی `MeyREN` رو انتخاب کنید.

### ۳. روشن کردن برنامه و ساخت آدرس

* روی دکمه **Deploy** کلیک کنید. ریلوی به صورت خودکار پروژه رو آماده و روشن میکنه (حدود ۱ تا ۲ دقیقه طول میکشه).
* وقتی کارش تموم شد و چراغش سبز شد، برید توی تب **Settings** (بالای صفحه پروژه در ریلوی)، یکم بیاید پایین‌تر و توی بخش **Networking** روی دکمه **Generate Domain** کلیک کنید تا آدرس اختصاصی پنل شما ساخته بشه.

### ۴. ورود به پنل مدیریت

* آدرسی که ریلوی بهتون داد رو کپی کنید و حتماً عبارت `/login` رو بذارید آخرش. (مثلاً اینطوری میشه: `https://your-app.up.railway.app/login`).
* حالا این آدرس رو توی مرورگر باز کنید.
* رمز عبور پیش‌فرض شما برای ورود کلمه `admin` هست.
* ⚠️ **خیلی مهم:** به محض اینکه برای اولین بار وارد پنل شدید، حتماً حتماً برید به بخش **Security** و رمز عبورتون رو عوض کنید تا امنیت پنلتون حفظ بشه.

## 🎯 چطوری از پنل استفاده کنیم؟

### ساختن لینک VLESS جدید

1. وارد پنل بشید.
2. روی دکمه **+ Add** کلیک کنید.
3. یه نام دلخواه برای کاربر بنویسید (مثلاً: `Ali`).
4. سقف حجمی که می‌تونه مصرف کنه رو مشخص کنید (مثلاً: `1` یعنی ۱ گیگابایت). اگه عدد رو `0` بذارید یعنی حجمش نامحدود میشه.
5. روی دکمه **Create** کلیک کنید.
6. حالا لینک ساخته شده؛ می‌تونید متنش رو کپی کنید یا QR Code اون رو با گوشی اسکن کنید.

### وارد کردن لینک در برنامه‌ها

1. برنامه **v2rayNG** یا **NekoBox** رو باز کنید.
2. لینکی که از پنل کپی کردید رو توی برنامه پیست (Paste) یا همون وارد کنید (از طریق گزینه پلاس یا Import از کلیپ‌بورد).
3. روی لینکی که اضافه شده بزنید و دکمه دایره‌ای اتصال رو فعال کنید.
4. 🎉 مبارکه! حالا به اینترنت آزاد دسترسی دارید.

### مدیریت و کنترل لینک‌ها

* 🔍 **جستجو:** اگه تعداد لینک‌هاتون زیاد شد، می‌تونید اسم یا کد (UUID) کاربر رو سرچ کنید تا راحت پیداش کنید.
* 🏷️ **فیلتر:** می‌تونید کاربران رو بر اساس اینکه الان فعال هستن یا حجمشون تموم شده و غیرفعال شدن جدا کنید.
* ⚡ **روشن/خاموش کردن:** با زدن روی دکمه Toggle (سوئیچ سبز رنگ)، می‌تونید بدون حذف کردن لینک، دسترسی اون کاربر رو موقتاً قطع یا وصل کنید.
* 📊 **دیدن مصرف:** حجم دقیق مصرفی هر کاربر به همراه یه نوار پیشرفت گرافیکی بهتون نشون داده میشه.
* 🗑️ **حذف:** هر وقت خواستید می‌تونید لینک‌های قدیمی یا بدون استفاده رو کاملاً حذف کنید.
