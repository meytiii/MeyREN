# 🚀 MeyREN Gateway

تونل VLESS over WebSocket با پنل مدیریت حرفه‌ای و دیتابیس ماندگار (SQLite)

---

## 📌 نحوه کارکرد

نیازی به سرور VPS ندارید. فقط کافیه پروژه را از [مخزن MeyREN](https://www.google.com/search?q=https://github.com/meytiii/MeyREN) فورک کنید و روی Railway دیپلوی کنید.

بعد از دیپلوی، یک آدرس دامنه اختصاصی از Railway بهتون داده می‌شود که پنل مدیریت شماست.

### خلاصه نحوه استفاده:

* 📱 اپلیکیشن v2rayNG یا NekoBox را نصب کنید


* 🔗 لینک VLESS را از پنل کپی کنید


* 📲 لینک را در اپلیکیشن وارد کنید


* ✅ اتصال برقرار شد!



---

## ⚡ شروع سریع (دیپلوی روی Railway)

### ۱. فورک کنید

* روی دکمه **Fork** بالای صفحه مخزن کلیک کنید.



### ۲. اتصال به Railway

* به سایت [Railway.app](https://railway.app) بروید و با اکانت GitHub خود وارد شوید.
* روی **+ New Project** کلیک کنید.
* گزینه **Deploy from GitHub repo** را انتخاب کرده و مخزن فورک شده خود (`MeyREN`) را انتخاب کنید.

### ۳. تنظیم متغیرهای محیطی (Environment Variables)

قبل از زدن دکمه دیپلوی، حتماً روی دکمه **Add Variables** کلیک کنید و متغیرهای زیر را وارد کنید تا پنل شما امنیت داشته باشد و کراش نکند:

* `ADMIN_PASSWORD`: رمز عبور دلخواه خود برای ورود به پنل (مثال: `mysecretpass`)
* `SECRET_KEY`: یک متن تصادفی طولانی برای امنیت هش کردن رمزها (مثال: `supersecretkey123456`)

### ۴. دیپلوی و دریافت دامنه

* روی **Deploy** کلیک کنید. Railway به صورت خودکار پروژه را می‌سازد و اجرا می‌کند.
* پس از اتمام دیپلوی، به تب **Settings** در محیط پروژه بروید و در بخش **Networking** روی **Generate Domain** کلیک کنید تا لینک پنل شما ساخته شود.

### ۵. وارد پنل شوید

* آدرس دریافتی از Railway را باز کنید (مثال: `[https://your-app.up.railway.app/login](https://your-app.up.railway.app/login)`).
* با رمز عبوری که در بخش متغیرها تنظیم کردید (`ADMIN_PASSWORD`) وارد شوید.

## 🎯 استفاده از پنل

### ساخت لینک VLESS

1. وارد پنل شوید


2. روی **+ Add** کلیک کنید


3. یک نام وارد کنید (مثلاً: `Ali`)


4. حجم مصرفی را مشخص کنید (مثلاً: `1` GB)


5. روی **Create** کلیک کنید


6. لینک VLESS را کپی کنید یا QR Code را اسکن کنید



### اتصال با اپلیکیشن

1. اپلیکیشن **v2rayNG** یا **NekoBox** را نصب کنید


2. لینک VLESS را وارد کنید


3. اتصال را فعال کنید


4. 🎉 تمام! حالا می‌توانید از اینترنت آزاد استفاده کنید



### مدیریت لینک‌ها

* 🔍 **جستجو:** در بخش Inbounds می‌توانید لینک‌ها را جستجو کنید


* 🏷️ **فیلتر:** لینک‌ها را بر اساس وضعیت (فعال/غیرفعال) فیلتر کنید


* ⚡ **فعال/غیرفعال:** با کلیک روی دکمه Toggle لینک را فعال یا غیرفعال کنید


* 📊 **مصرف:** مصرف حجم هر لینک را ببینید


* 🗑️ **حذف:** لینک‌های بدون استفاده را حذف کنید
